Lala Alerts Demo

This alert plugin is for popping up alerts of different statuses (currently 4 right now). But feel free to add more if you'd like. I took the colours for Twitter's Bootstrap alerts. I used vanilla js for this alert plugin so it should run pretty fast. Feel free to optimize it even more.

Click on one of the buttons to see an alert pop-up. By default, the alerts exist for 8 seconds before it disappears (destroyed in memory and in the DOM).
